# Necessary info:
1. Creating .env file and adding these following fields to run script:
REPEAT_TIME=
TYPING_STRING=
MOUSE_X_POSITION=
MOUSE_Y_POSITION=
2. Installing node and running "npm install" to install all necessary packages
# How to run this script?
1. Using node execution:
node path-to-script/index.js
2. Using your ide and run the command:
npm start
