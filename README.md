# Necessary info:
1. Creating .env file and adding these following fields to run script (you can custom all of these params):
REPEAT_TIME=5000
TYPING_STRING=Hello World
MOUSE_X_POSITION=50
MOUSE_Y_POSITION=0
KEY_TAP=down
NEED_MOVE_MOUSE=true
NEED_SCROLL_MOUSE=true
NEED_CLICK_MOUSE=true
NEED_TYPING=true
2. Installing node and running "npm install" to install all necessary packages
# How to run this script?
1. Using node execution:
node path-to-script/index.js
2. Using your ide and run the command:
npm start
